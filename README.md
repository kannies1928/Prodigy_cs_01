# PRODIGY_CS_01 - Temperature Converter

## Task Description
Python program to convert temperatures between Celsius, Fahrenheit, and Kelvin scales.
User enters temperature value + original unit, program outputs the other two units.

## How to Run
```bash
python task_01.py

Enter temperature value: 25
Enter original unit - C/F/K: C

25.0°C = 77.00°F
25.0°C = 298.15K

## Formulas Used
- Celsius to Fahrenheit: (C × 9/5) + 32
- Celsius to Kelvin: C + 273.15
- Fahrenheit to Celsius: (F - 32) × 5/9
- Kelvin to Celsius: K - 273.15
